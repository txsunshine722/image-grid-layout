# Image Grid Layout

A responsive image grid layout built with HTML and CSS.

## Project Links
 
- Project URL: https://roadmap.sh/projects/image-grid
- GitHub Repository: https://github.com/txsunshine722/image-grid-layout

## About This Project

This project is focused on building an image grid layout using only HTML and CSS.

The layout displays multiple images in a clean structure and adapts across screen sizes.

## Key Learning Objectives

1. Grid Structure
    - Understand the fundamentals of CSS Grid, including grid containers, rows, columns, and item placement.

2. Responsive Web Design
    - Learn how to make the layout adapt to different screen sizes by changing columns or layout behavior based on viewport width.

3. Alignment and Spacing
    - Explore alignment tools and techniques to create a clean and visually appealing image layout.

## Tech Stack

- HMTL5
- CSS3

## How to Run Locally

1. Clone the repository:
    git clone https://github.com/txsunshine722/image-grid-layout.git

2. Opent the project folder in VS Code.

3. Open index.html in your browser (or use Live Server).
